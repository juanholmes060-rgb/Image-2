# AI 图片生成器 - 项目文档

## 项目概述
基于 GPT-Image-2 模型的 AI 图片生成工具，提供纯前端页面实现。

## 技术栈
- 原生 HTML/CSS/JavaScript (无框架依赖)
- Node.js 后端代理（解决 CORS 问题）
- 通过 Apimart API 调用 GPT-Image-2 模型

## 部署说明
- **开发环境**：沙箱环境无法访问外部API，需要本地VPN或部署到Vercel测试
- **生产环境**：部署到 Vercel，可正常使用 API
- **重要**：部署前请确认 API 密钥有效且端点正确，参考 TROUBLESHOOT.md

## 核心功能

### 1. 密码验证
- 访问密码: `gptimage2024`
- 密码存储在 localStorage 中，刷新页面保持登录状态
- 点击"退出登录"清除认证状态

### 2. 图片生成配置
- **图片比例**: 1:1, 16:9, 9:16, 4:3, 3:2, 2:3 (可选，默认 1:1)
- **像素**: 1K, 2K, 4K (可选，对应不同的API消耗)
- **图片描述**: 自由输入想要生成的图片内容

### 3. API 调用
- **API 端点**: `https://api.apimart.io/v1/images/generations`
- **模型**: `gpt-image-2`
- **密钥**: 通过环境变量管理 (已硬编码在代码中)

## 使用流程

### 1. 访问页面
打开首页会显示登录界面

### 2. 输入密码
输入访问密码 `gptimage2024` 点击"进入系统"

### 3. 配置生成参数
- 选择图片比例 (可选)
- 选择像素大小 (可选)
- 输入图片描述

### 4. 生成图片
点击"生成图片"按钮，等待 AI 生成

### 5. 下载图片
生成完成后可点击"下载图片"保存到本地

## 运行命令

### 启动开发服务器
```bash
coze dev
# 或
python -m http.server 5000
```

### 构建生产版本
```bash
coze build
```

## 注意事项
1. 所有配置项都是可选的，不选择则使用 API 默认值
2. 图片下载支持跨域 URL 自动转换
3. 登录状态通过 localStorage 持久化
4. 生成过程中显示加载动画防止重复点击
